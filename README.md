고요하고 평화로운 분위기의 시골을 배경으로 한 힐링 게임입니다.

이 게임은 플레이어가 주변의 NPC과 대화를 나누는 것이 주요한 힐링 요소 중 하나입니다.

누구나 자유롭게 이용할 수 있어요! 언제든지 와서 휴식을 취하고 마음을 편안하게 하세요!
